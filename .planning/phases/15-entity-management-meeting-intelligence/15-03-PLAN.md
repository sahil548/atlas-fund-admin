Props: `entity: any` (full entity object from API)

A banner/card component that appears on the entity overview tab when `formationStatus` is `"FORMED"` or `"REGISTERED"`.

**Context-aware checklist logic:**

Determine if entity is deal-linked: check if `entity.dealEntities?.length > 0` or if entity was created as part of a deal.

Deal-linked vehicles get these steps:
1. "File SEC exemptions" -> links to `?tab=regulatory` -> done if `regulatoryFilings` JSON has any filing records
2. "Add investors & commitments" -> links to `?tab=investors` -> done if `entity.commitments?.length > 0`
3. "Issue capital call" -> links to `?tab=capital` -> done if `entity.capitalCalls?.length > 0`
4. "Verify funding received" -> links to `/transactions` -> done if any capital call has status "FUNDED"
5. "Close on deal" -> links to `/deals` -> done: false (always unchecked, final step)

Standalone vehicles get these steps:
1. "Add investors" -> links to `?tab=investors` -> done if `entity.commitments?.length > 0`
2. "Configure waterfall" -> links to `?tab=waterfall` -> done if `entity.waterfallTemplate` exists
3. "Upload governing docs" -> links to `?tab=documents` -> done if `entity.documents?.length > 0`
4. "Connect QuickBooks" -> links to `?tab=accounting` -> done if `entity.accountingConnection?.syncStatus === "CONNECTED"`
5. "Add regulatory filings" -> links to `?tab=regulatory` -> done if `regulatoryFilings` has filing records

**UI design:**
- Rounded card with light blue/indigo background: `bg-indigo-50 dark:bg-indigo-900/20 border border-indigo-200 dark:border-indigo-800 rounded-xl p-4`
- Header: "What's Next" with a checklist icon, and a progress indicator (e.g., "3/5 complete")
- Each item is a row with: checkbox (checked/unchecked based on done state), label, and right-aligned arrow link
- Checked items have strikethrough and muted text
- Each label is a clickable link using `onClick={() => setTab(targetTab)}` pattern or Link component
- Include a dismiss/collapse button so GP can hide the banner after reviewing
