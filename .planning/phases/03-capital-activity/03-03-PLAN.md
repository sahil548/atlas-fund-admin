**Create `src/app/api/entities/[id]/metrics/route.ts`:**

GET endpoint that computes all metrics for an entity in real-time:
1. Fetch entity with commitments, capital calls (with line items), distributions (with line items), asset allocations (with assets + valuations)
2. Compute `totalCalled`: sum of all funded CapitalCallLineItem amounts for this entity
3. Compute `totalDistributed`: sum of all PAID DistributionLineItem netAmounts for this entity
4. Get `currentNAV`: call the NAV computation logic inline (same as /api/nav/[entityId])
5. Compute `totalCostBasis`: sum of asset allocation cost bases
6. Compute `totalFairValue`: sum of asset fair values * allocation %
7. Call `computeMetrics(totalCalled, totalDistributed, currentNAV.economicNAV, totalCostBasis, totalFairValue)` -> { tvpi, dpi, rvpi, moic }
8. Compute entity-level MOIC as weighted average: `sum(asset.fairValue * allocation%) / sum(asset.costBasis * allocation%)` — per user decision "MOIC at entity level (weighted average)"
9. Compute IRR using `xirr()`:
   - Capital call funding events as negative cash flows (outflows): `{ date: lineItem.paidDate, amount: -lineItem.amount }` for each funded line item
   - Distribution payments as positive cash flows (inflows): `{ date: distribution.distributionDate, amount: lineItem.netAmount }` for each PAID distribution line item
   - Current NAV as terminal positive cash flow: `{ date: new Date(), amount: currentNAV.economicNAV }`
10. Return:
```json
{
  "entityId": "...",
  "metrics": { "tvpi": 1.45, "dpi": 0.35, "rvpi": 1.10, "moic": 1.52, "irr": 0.128 },
  "inputs": { "totalCalled": ..., "totalDistributed": ..., "currentNAV": ..., "costBasis": ..., "fairValue": ... }
}
```
