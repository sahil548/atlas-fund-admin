    4. Create src/lib/computations/performance-attribution.ts:
       - computeAssetAttribution(assetId): Promise<AssetAttribution>
         - Queries: asset (with sourceDeal for projections), capital calls allocated to this asset's entities, distributions, income events, valuations
         - Computes actual metrics:
           - actualIRR: XIRR from cash flows (capital calls as negatives, distributions + current fair value as positives)
           - actualMOIC: fairValue / costBasis
           - actualTotalReturn: (fairValue + totalDistributions - costBasis) / costBasis
           - totalIncome: sum of income events
           - unrealizedGain: fairValue - costBasis
           - realizedGain: totalDistributions - (costBasis * pctReturned)
         - Extracts projected metrics:
           - From asset.projectedIRR, asset.projectedMultiple, asset.projectedMetrics
           - Falls back to sourceDeal.dealMetadata if asset projections not set
         - Returns { actual: { irr, moic, totalReturn, income, unrealizedGain, realizedGain }, projected: { irr, multiple, classSpecific: {...} }, variance: { irrDelta, moicDelta }, assetInfo: { name, assetClass, entityName, costBasis, fairValue } }
       - computeEntityAttribution(entityId): Promise<EntityAttribution>
         - For each asset allocated to the entity:
           - Compute asset attribution (above)
           - Compute contribution weight: asset costBasis / total entity costBasis
           - Compute contribution to fund IRR: weightedContribution = weight * assetIRR
         - Returns { entityMetrics: { totalIRR, totalTVPI, totalCostBasis, totalFairValue }, assets: AssetAttribution[], rankedByContribution: [{ assetName, contributionPct, irr, moic }] }
  </action>
