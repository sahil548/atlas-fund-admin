- Uses Modal from `@/components/ui/modal`
- Form fields: Exit Date (date input, required), Exit Proceeds (number input with $ prefix, required), Exit Notes (textarea, optional)
- Live preview section (computed on every render, NO useEffect):
  ```typescript
  const previewMoic = exitProceeds ? (parseFloat(exitProceeds) / asset.costBasis) : null;
  const previewGainLoss = exitProceeds ? parseFloat(exitProceeds) - asset.costBasis : null;
  const holdPeriodDays = exitDate && asset.entryDate
    ? Math.floor((new Date(exitDate).getTime() - new Date(asset.entryDate).getTime()) / (1000 * 60 * 60 * 24))
    : null;
  ```
