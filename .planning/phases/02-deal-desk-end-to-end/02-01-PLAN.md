  Add ALL new models and fields needed for Phase 2. This is the single schema migration for the entire phase.

  **A. Kill/Revive fields on Deal:**
  - `killReason String?` -- stores reason category (Pricing, Risk, Timing, Sponsor, Other)
  - `killReasonText String?` -- stores free text explanation
  - `previousStage DealStage?` -- stores stage before kill (for revive)
  - `dealMetadata Json?` -- AI-extracted structured data (size, target return, terms, structure)

  **B. DealEntity junction table (new model):**
  ```
  model DealEntity {
    id                String   @id @default(cuid())
    dealId            String
    entityId          String
    allocationPercent Float?
    role              String?  // "Lead Entity", "Co-Investor"
    createdAt         DateTime @default(now())

    deal   Deal   @relation(fields: [dealId], references: [id], onDelete: Cascade)
    entity Entity @relation(fields: [entityId], references: [id])

    @@unique([dealId, entityId])
    @@index([dealId])
    @@index([entityId])
  }
  ```
  - Add `dealEntities DealEntity[]` to Deal model (keep existing entityId for backward compat)
  - Add `dealEntities DealEntity[]` to Entity model

  **C. ClosingChecklist additions:**
  - `fileUrl String?` -- Vercel Blob URL for attached file
  - `fileName String?` -- display name of attached file
  - `isCustom Boolean @default(false)` -- distinguishes template vs custom items

  **D. Asset source tracking:**
  - Add `sourceDealId String?` to Asset model
  - Add `sourceDeal Deal? @relation("AssetSourceDeal", fields: [sourceDealId], references: [id])`
  - Add `sourceAssets Asset[] @relation("AssetSourceDeal")` to Deal model

  **E. DDWorkstream PM enhancements:**
  - Add `assigneeId String?` to DDWorkstream
  - Add `priority String? @default("MEDIUM")` -- High/Medium/Low
  - Add `dueDate DateTime?` to DDWorkstream
  - Add relation: `assignee User? @relation("WorkstreamAssignee", fields: [assigneeId], references: [id])`
  - Add `assignedWorkstreams DDWorkstream[] @relation("WorkstreamAssignee")` to User model

  **F. DDWorkstreamComment (new model):**
  ```
  model DDWorkstreamComment {
    id           String   @id @default(cuid())
    workstreamId String
    authorId     String
    content      String
    parentId     String?  // for threading
    createdAt    DateTime @default(now())
    updatedAt    DateTime @updatedAt

    workstream DDWorkstream @relation(fields: [workstreamId], references: [id], onDelete: Cascade)
    author     User         @relation("WorkstreamCommentAuthor", fields: [authorId], references: [id])
    parent     DDWorkstreamComment?  @relation("CommentThread", fields: [parentId], references: [id])
    replies    DDWorkstreamComment[] @relation("CommentThread")

    @@index([workstreamId])
  }
  ```
  - Add `comments DDWorkstreamComment[]` to DDWorkstream model
  - Add `workstreamComments DDWorkstreamComment[] @relation("WorkstreamCommentAuthor")` to User model

  **G. DDWorkstreamAttachment (new model):**
  ```
  model DDWorkstreamAttachment {
    id           String   @id @default(cuid())
    workstreamId String
    fileName     String
    fileUrl      String
    fileSize     Int?
    uploadedById String?
    createdAt    DateTime @default(now())

    workstream DDWorkstream @relation(fields: [workstreamId], references: [id], onDelete: Cascade)

    @@index([workstreamId])
  }
  ```
  - Add `attachments DDWorkstreamAttachment[]` to DDWorkstream model

  **H. Decision-making structures (IC config):**
  ```
  model DecisionStructure {
    id               String   @id @default(cuid())
    firmId           String
    name             String
    description      String?
    quorumRequired   Int      @default(1)
    approvalThreshold Int     @default(1)  // votes needed for approval
    createdAt        DateTime @default(now())
    updatedAt        DateTime @updatedAt

    firm    Firm               @relation(fields: [firmId], references: [id])
    members DecisionMember[]
    entities Entity[]          @relation("EntityDecisionStructure")

    @@index([firmId])
  }

  model DecisionMember {
    id          String   @id @default(cuid())
    structureId String
    userId      String
    role        String?  @default("VOTER")  // VOTER, OBSERVER
    createdAt   DateTime @default(now())

    structure DecisionStructure @relation(fields: [structureId], references: [id], onDelete: Cascade)
    user      User              @relation(fields: [userId], references: [id])

    @@unique([structureId, userId])
  }
  ```
  - Add `decisionStructures DecisionStructure[]` to Firm model
  - Add `decisionMemberships DecisionMember[]` to User model
  - Add `decisionStructureId String?` to Entity model
  - Add `decisionStructure DecisionStructure? @relation("EntityDecisionStructure", fields: [decisionStructureId], references: [id])` to Entity model
