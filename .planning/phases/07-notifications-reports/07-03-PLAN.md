       ```typescript
       export const GenerateReportSchema = z.object({
         entityId: z.string(),
         reportType: z.enum(["QUARTERLY", "CAPITAL_ACCOUNT_STATEMENT", "FUND_SUMMARY"]),
         period: z.string().optional(), // e.g., "Q4 2025" or "2025"
         investorId: z.string().optional(), // for investor-specific capital account statement
       });
       ```

    2. Create `src/app/api/reports/generate/route.ts`:
       - POST handler: parseBody(req, GenerateReportSchema)
       - Fetch entity data from Prisma (entity, commitments, capital calls, distributions, assets, NAV, metrics)
       - Based on reportType, assemble data object and import the correct React-PDF component
       - Use `renderToBuffer` from @react-pdf/renderer to generate PDF buffer
       - Upload to Vercel Blob: `put(`reports/${entityId}/${reportType}_${period}_${Date.now()}.pdf`, buffer, { access: "public", contentType: "application/pdf" })`
       - Create Document record: `prisma.document.create({ data: { name, category: reportType === "CAPITAL_ACCOUNT_STATEMENT" ? "STATEMENT" : "REPORT", fileUrl: blobUrl, fileSize: buffer.length, mimeType: "application/pdf", entityId } })`
       - Return { documentId, fileUrl, fileName }
       - IMPORTANT: this runs on Vercel serverless — must complete within 10s. @react-pdf/renderer is fast enough per user research
