    1. Add to `prisma/schema.prisma`:
       ```prisma
       model IntegrationConnection {
         id            String   @id @default(cuid())
         firmId        String
         provider      String   // "asana" | "notion" | "google_calendar" | "plaid"
         entityId      String?  // for per-entity integrations (Plaid)
         accessToken   String
         refreshToken  String?
         tokenExpiry   DateTime?
         externalAccountId String? // provider-specific account/workspace ID
         metadata      Json?    // provider-specific config (workspace ID, project ID, etc.)
         createdAt     DateTime @default(now())
         updatedAt     DateTime @updatedAt

         @@unique([firmId, provider, entityId])
       }

       model PlaidAccount {
         id              String   @id @default(cuid())
         connectionId    String
         accountId       String   // Plaid account_id
         name            String
         type            String   // checking, savings, etc.
         subtype         String?
         currentBalance  Float?
         availableBalance Float?
         lastSynced      DateTime?
         createdAt       DateTime @default(now())
         updatedAt       DateTime @updatedAt

         @@unique([connectionId, accountId])
       }
       ```
       Run `npx prisma generate`.

    2. Create `src/lib/integrations/asana.ts`:
       - Use raw fetch (no Asana SDK — same pattern as QBO, keeps bundle small)
       - Export `AsanaClient` class:
         - constructor(connection: IntegrationConnection)
         - `static getAuthUrl(firmId)`: Asana OAuth2 URL
         - `static exchangeCode(code, firmId)`: token exchange
         - `async refreshTokenIfNeeded()`: refresh before expiry
         - `async listWorkspaces()`: GET /api/1.0/workspaces
         - `async syncTasksToAsana(tasks: AtlasTask[])`: create/update tasks in configured Asana project
         - `async syncTasksFromAsana(projectGid)`: fetch Asana tasks, return for Atlas import
       - Bidirectional: Atlas tasks -> Asana tasks, Asana tasks -> Atlas tasks
       - Map Atlas task fields (title, status, assignee, dueDate) <-> Asana task fields (name, completed, assignee, due_on)

    3. Create `src/lib/integrations/notion.ts`:
       - Use raw fetch to Notion API (Authorization: Bearer token, Notion-Version header)
       - Export `NotionClient` class:
         - constructor(connection)
         - `static getAuthUrl(firmId)`: Notion OAuth URL
         - `static exchangeCode(code, firmId)`: token exchange
         - `async listDatabases()`: search for databases the integration has access to
         - `async syncToNotion(data)`: create/update pages in configured Notion database
         - `async syncFromNotion(databaseId)`: read pages from Notion, return for Atlas import
       - Sync Atlas deal data <-> Notion database rows (name, stage, status, etc.)

    4. Create `src/lib/integrations/plaid.ts`:
       - Install `plaid` SDK: `npm install plaid` (Plaid requires their SDK for Link Token)
       - Export functions:
         - `createLinkToken(firmId, entityId)`: POST to /link/token/create with required fields
         - `exchangePublicToken(publicToken, firmId, entityId)`: exchanges Plaid Link public_token for access_token
         - `getAccounts(connection)`: GET /accounts/get — returns balances
         - `getTransactions(connection, startDate, endDate)`: GET /transactions/get — returns recent transactions
       - Per-entity connection (mirrors QBO pattern per user decision)

    5. Create `src/lib/integrations/google-calendar.ts`:
       - Use raw fetch to Google Calendar API v3
       - Export `GoogleCalendarClient` class:
         - constructor(connection)
         - `static getAuthUrl(firmId)`: Google OAuth URL (scope: calendar.events)
         - `static exchangeCode(code, firmId)`: token exchange
         - `async refreshTokenIfNeeded()`: refresh before expiry
         - `async listEvents(calendarId, timeMin, timeMax)`: GET /calendars/{id}/events
         - `async createEvent(calendarId, event)`: POST /calendars/{id}/events
         - `async syncMeetings(meetings)`: push Atlas meetings to Google Calendar
         - `async syncTaskDueDates(tasks)`: create calendar events for task due dates
       - Bidirectional: Google events -> Atlas meetings, Atlas meetings -> Google events
